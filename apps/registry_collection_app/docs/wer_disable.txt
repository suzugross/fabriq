【Windowsエラー報告を無効化】

Windows Error Reporting の Disabled = 1 に設定することで、
アプリクラッシュ時のエラーレポート送信ダイアログおよびMicrosoftへの
クラッシュデータ送信を無効化します。

■ キッティングでの用途
  社内環境でのプライバシーポリシー遵守のため、または
  エラーダイアログの表示を抑制したい場合に使用します。

■ 注意事項
  この設定を有効にすると、Microsoftへのクラッシュレポートが送信されなくなります。
  一方で、社内のWER（Windows Error Reporting）サーバーへの送信は別途設定が必要です。

■ 確認方法
  設定 → プライバシーとセキュリティ → 診断とフィードバック

■ 既定値
  Disabled = 0（エラー報告を有効にする）
