【高速スタートアップを無効化】

Session Manager\Power の HiberbootEnabled = 0 に設定することで、
Windowsの「高速スタートアップ」機能を無効化します。

■ 高速スタートアップとは
  シャットダウン時にカーネルセッションをハイバネーション状態で保存し、
  次回起動時間を短縮するWindows 8以降の機能です。

■ 無効化が推奨されるケース
  - 起動時にグループポリシーが正しく適用されない
  - 再起動後もドライバの変更が反映されない
  - BitLockerを使用しており、PCR値の変動で解錠に失敗する
  - デュアルブート環境でLinux側からWindowsパーティションへアクセスしたい
  - キッティング後に確実にフル初期化起動させたい

■ 確認方法
  コントロールパネル → 電源オプション → 電源ボタンの動作を選択する
  → 「高速スタートアップを有効にする（推奨）」のチェックボックス

■ 既定値
  HiberbootEnabled = 1（高速スタートアップを有効にする）
