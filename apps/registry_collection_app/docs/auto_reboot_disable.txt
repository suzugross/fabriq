【システム障害時の自動再起動を無効化】

CrashControl の AutoReboot = 0 に設定することで、
BSOD（ブルースクリーン）発生時に自動再起動せず、エラー画面で停止するようになります。

■ キッティングでの用途
  クラッシュの原因を調査・記録するため、BSOD画面のストップコードを
  確認できるようにします。キッティング後の検証フェーズで有用です。

■ 確認方法
  コントロールパネル → システム → システムの詳細設定
  → 起動と回復 → 「自動的に再起動する」のチェックボックス

■ 関連設定
  同じ CrashControl キー配下に以下の設定も存在します：
    DumpFile     = ダンプファイルの保存先（既定: %SystemRoot%\MEMORY.DMP）
    CrashDumpMode = ダンプの種類（0=なし, 1=完全, 2=カーネル, 3=ミニ, 7=自動）

■ 既定値
  AutoReboot = 1（自動再起動する）
