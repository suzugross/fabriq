========================================
SPI Configuration (SystemParametersInfo) - Guide
========================================

■ 概要
  Windows の SystemParametersInfo API を使用して、レジストリ直接書き込みでは
  安全に制御できない設定を CSV 駆動で変更するモジュール。
  設定は現在のユーザーセッションに即時反映される。

■ なぜこのモジュールが必要か
  一部の Windows 設定は UserPreferencesMask（REG_BINARY のビットフラグ）に
  格納されており、レジストリを直接書き換えると関係ないビットを壊すリスクがある。
  SystemParametersInfo API は指定した項目のビットのみを安全に更新する。

  reg_hkcu_config で制御できる項目はそちらを使用し、
  このモジュールは SPI 経由でしか安全に変更できない項目に使用する。

■ Default Profile への適用
  SPI は現在のユーザーセッションにのみ反映される。
  新規ユーザー向けの Default Profile 設定は reg_hkcu_config を併用すること。

■ CSV カラム説明 (spi_list.csv)

  Enabled    : 1=実行, 0=スキップ
  SpiAction  : SET 用の SPI 定数（16進数、例: 0x1043）
  ValueMode  : 値の渡し方（下記3種）
  Value      : 設定値（bool の場合 0=OFF, 1=ON）
  Description: 表示用の説明文
  Segment    : プロファイル用フィルタ（空=全対象）

■ ValueMode の種類

  bool       SET 時 pvParam に 0 または 1 を渡す。
             視覚効果のON/OFF等。
             例: 0x1043, bool, 0 → アニメーション OFF

  uiParam    SET 時 uiParam に整数値を渡す。
             キーボード速度、キーボード遅延等。
             例: 0x000B, uiParam, 31 → キーリピート速度 最速

  pvParam    SET 時 pvParam に整数値を渡す。
             マウス速度、メニュー表示遅延等。
             例: 0x0071, pvParam, 10 → マウス速度 10

■ 冪等性
  各設定の GET 値を取得し（GET action = SET action - 1）、
  目標値と一致する場合はスキップする。

========================================
SPI 定数 対応表
========================================

--- 視覚効果 (UserPreferencesMask) ---
  全て ValueMode = bool で制御。0=OFF, 1=ON。

  SpiAction  | 設定名
  -----------+-------------------------------------------
  0x1043     | Windows内のアニメーション コントロールと要素
  0x1025     | ウィンドウの下に影を表示する
  0x1005     | コンボボックスをスライドして開く
  0x1017     | ヒントをフェードまたはスライドで表示する
  0x101B     | マウスポインターの下に影を表示する
  0x1003     | メニューをフェードまたはスライドして表示する
  0x1015     | メニュー項目をクリック後にフェードアウトする
  0x1007     | リストボックスを滑らかにスクロールする

--- UI 制御 (bool) ---
  0x1001     | マウスホバーでウィンドウをアクティブ化
  0x100D     | アクティブ追従時の前面移動
  0x0011     | スクリーンセーバー有効/無効
  0x0060     | ダイアログの既定ボタンにスナップ
  0x1041     | 透過表示の無効化

--- マウス (pvParam) ---
  0x0071     | マウス速度 (1-20)

--- キーボード (uiParam) ---
  0x000B     | キーリピート速度 (0=遅い - 31=速い)
  0x0017     | キーリピート遅延 (0=短い - 3=長い)

--- タイマー (pvParam) ---
  0x006B     | メニュー表示遅延 (ミリ秒、既定400)
  0x0067     | マウスホバー判定時間 (ミリ秒)
  0x2001     | フォアグラウンドロック時間 (ミリ秒)

--- スクリーンセーバー (uiParam) ---
  0x000F     | スクリーンセーバー待ち時間 (秒)

========================================

  視覚効果 パフォーマンス重視（8項目すべて OFF）:
    Enabled,SpiAction,ValueMode,Value,Description,Segment
    1,0x1043,bool,0,Windows内のアニメーション コントロールと要素,
    1,0x1025,bool,0,ウィンドウの下に影を表示する,
    1,0x1005,bool,0,コンボボックスをスライドして開く,
    1,0x1017,bool,0,ヒントをフェードまたはスライドで表示する,
    1,0x101B,bool,0,マウスポインターの下に影を表示する,
    1,0x1003,bool,0,メニューをフェードまたはスライドして表示する,
    1,0x1015,bool,0,メニュー項目をクリック後にフェードアウトする,
    1,0x1007,bool,0,リストボックスを滑らかにスクロールする,

  マウス速度 + キーボード速度の設定:
    Enabled,SpiAction,ValueMode,Value,Description,Segment
    1,0x0071,pvParam,10,マウス速度(中速),
    1,0x000B,uiParam,31,キーリピート速度(最速),
    1,0x0017,uiParam,0,キーリピート遅延(最短),
