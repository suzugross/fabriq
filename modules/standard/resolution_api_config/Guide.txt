■ 概要
ディスプレイの解像度を変更します。
Windows APIで即時反映されるため、再起動は不要です。

■ 設定ファイル
[resolution_list.csv]
  Enabled     : 有効フラグ（1=実行, 0=スキップ）
  Width       : 横幅（ピクセル）
  Height      : 高さ（ピクセル）
  Description : 説明（表示用）

[記載例]
  1,1920,1080,Full HD
  0,2560,1440,QHD
  0,1366,768,WXGA

■ 動作の流れ
1. resolution_list.csv を読み込み
2. 変更先の解像度を表示
3. 実行確認（Y/N）後、解像度を変更
