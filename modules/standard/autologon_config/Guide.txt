■ 概要
Windowsの自動ログオンを設定します。
AutoLogonCount=1 により、次回の1回のみ自動ログオンし、
ログオン後に資格情報は自動的にクリアされます。

■ 設定ファイル
[autologon_list.csv]
  Enabled     : 有効フラグ（1=実行, 0=スキップ）
  No          : 識別番号（選択画面での表示用）
  User        : ユーザー名
  Password    : パスワード
  Domain      : ドメイン名（ローカルユーザーの場合は空欄）
  Description : 説明（表示用）

[記載例]
  1,1,admin01,P@ssw0rd,,ローカル管理者
  1,2,user01,Pass1234,example.local,ドメインユーザー

■ ユーザーの選択
・有効なエントリが1件の場合は自動選択されます
・複数ある場合はNo番号で選択します
・プロファイル実行時は __AUTO_to_ユーザー名__ で自動指定できます

■ 動作の流れ
1. autologon_list.csv を読み込み
2. 対象ユーザーを選択・表示（パスワードはマスク表示）
3. 実行確認（Y/N）
4. レジストリ（Winlogon）に自動ログオン情報を設定
