■ 概要
ディスプレイのDPIスケーリング（拡大率）を設定します。
Windows APIで即時反映されるため、再起動は不要です。

■ 設定ファイル
[dpi_list.csv]
  Enabled      : 有効フラグ（1=実行, 0=スキップ）
  MonitorIndex : モニター番号（0=プライマリ, 1=セカンダリ, ...）
  ScalePercent : 拡大率（例: 100, 125, 150, 175, 200）
  Description  : 説明（表示用）

[記載例]
  1,0,150,プライマリモニターを150%に設定
  1,1,125,セカンダリモニターを125%に設定

■ 動作の流れ
1. dpi_list.csv を読み込み
2. 対象モニターと拡大率を表示
3. 実行確認（Y/N）後、DPIを変更
