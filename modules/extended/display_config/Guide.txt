■ 概要
ディスプレイの解像度をレジストリ経由で設定します。
standard の resolution_api_config（即時反映）とは異なり、再起動後に反映されます。

■ 設定ファイル
[display_list.csv]
  Enabled     : 有効フラグ（1=実行, 0=スキップ）
  HardwareID  : ディスプレイのハードウェアID（AUTO=自動検出、空欄=対話選択）
  Width       : 横幅（ピクセル）
  Height      : 高さ（ピクセル）
  Description : 説明（表示用）

[記載例]
  1,AUTO,1920,1080,プライマリディスプレイ自動検出
  1,CMN14D40,1920,1080,内蔵14インチFHD
  1,LEN61CC,2560,1440,外部モニター QHD

■ HardwareID の指定方法
・AUTO : プライマリディスプレイを自動検出（複数接続時は対話選択にフォールバック）
・具体的なID : デバイスマネージャーのモニターのハードウェアIDを指定
・空欄 : 実行時に対話形式で選択

■ 前提条件
・管理者権限が必要です

■ 注意事項
・変更の反映には再起動が必要です

■ 動作の流れ
1. display_list.csv を読み込み
2. 対象ディスプレイと現在の解像度を表示
3. 実行確認（Y/N）後、レジストリに書き込み
